Application fullstack qui sert de site vitrine pour l'association Réunies, dédiée au soutien et à la promotion des femmes entrepreneures. L'application permet aux utilisatrices de s'inscrire, de se connecter, et de rechercher d'autres membres en fonction de critères avancés (activités, régions, villes) via une interface web intuitive. Les fonctionnalités sont rendues possibles grâce à des endpoints API REST conçus pour gérer l'inscription, la connexion et la recherche dynamique. Le frontend a été déployé sur Netlify, tandis que le backend est hébergé sur Northflank.

Emoji Search

Stack technique

HTML

CSS

JavaScript

React

Vite

ESlint

L'application permet à l'utilisateur de rechercher un émoji en saisissant son nom dans un champ de recherche. Par défaut, une liste de 20 émojis est affichée lorsqu'aucune recherche n'est effectuée. Les émojis s'affichent dynamiquement au fur et à mesure de la saisie de l'utilisateur. Si aucune correspondance n'est trouvée, un message informe l'utilisateur qu'aucun émoji ne correspond à sa recherche.
